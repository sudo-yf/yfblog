## 📝 核心知识点速览

### 四大变形类型

| 变形形式 | 刚度表达式 | 强度公式 | 变形公式 |
|---------|-----------|---------|---------|
| 拉压 | $EA$ | $\sigma = \frac{F}{A}$ | $\Delta L = \frac{FL}{EA}$ |
| 扭转 | $GI_p$ | $\tau = \frac{T}{W_p}$ | $\varphi = \frac{Tl}{GI_p}$ |
| 弯曲 | $EI$ | $\sigma = \frac{M}{W_z}$ | $w'' = \frac{M}{EI}$ |
| 剪切 | - | $\tau = \frac{F_s}{A}$ | - |

### 四大强度理论

1. **最大拉应力理论**（第一）：$\sigma_1 \le [\sigma]$ （脆性材料）
2. **最大伸长线应变理论**（第二）：$\sigma_1 - \mu(\sigma_2 + \sigma_3) \le [\sigma]$
3. **最大剪应力理论**（第三）：$\sigma_1 - \sigma_3 \le [\sigma]$ （塑性材料）
4. **畸变能理论**（第四）：$\sqrt{\frac{1}{2}[(\sigma_1-\sigma_2)^2 + (\sigma_2-\sigma_3)^2 + (\sigma_3-\sigma_1)^2]} \le [\sigma]$

## ⚠️ 易错提醒

- ❌ 扭转变形用 $I_p$，不是 $W_p$
- ❌ 动荷系数公式：根号内**没有平方**
- ❌ 空心圆截面系数：$\alpha$ 是 **4次方**
- ❌ T型梁必须分别校核拉、压应力
- ❌ 形心公式：$y_c = \frac{\sum A_i y_i}{\sum A_i}$

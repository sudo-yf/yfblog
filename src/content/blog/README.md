# Blog Content Structure

## Categories

### Study (课内学习)
存放课内学习相关的笔记和文章

### Learning (课外学习)
存放课外学习、技术探索相关的内容

### Recommendations (影音书推荐)
存放电影、音乐、书籍推荐

### Life (日记/杂谈)
按日期记录的日记，格式：`YYYY-MM-DD.md`

## File Structure
```
src/content/blog/
├── Study/
│   └── [topic-name]/
│       ├── index.md
│       └── assets/
├── Learning/
│   └── [topic-name]/
│       ├── index.md
│       └── assets/
├── Recommendations/
│   └── [item-name]/
│       ├── index.md
│       └── assets/
└── Life/
    └── YYYY-MM-DD.md
```

## Diary Template
每日日记模板包含：
- 工作流记录（上午、下午、晚上）
- 今日感悟
- 待办事项
- 学习记录
